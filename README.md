# Belkacem Raber — Portfolio Ecosystem 🚀

Bienvenue sur le dépôt central de mes travaux de conception et d'architecture digitale. Ce repository héberge mon projet phare : **The Grand Voyage**.

## 🪐 [The Grand Voyage — Projet Principal](./the-grand-voyage-final/)

Une expérience immersive et cinématique conçue comme un voyage vertical à travers le cosmos et les profondeurs terrestres. Ce portfolio démontre une synergie entre **design haute-fidélité**, **performances natives** et **cybersécurité**.

### Points Forts du Projet :
- **Expérience Multilingue** : Système i18n complet (FR, EN, AR) avec support natif du mode **RTL** (Right-to-Left) pour l'arabe.
- **UI Orbitale** : Sélecteur de langue exclusif inspiré des mécanismes célestes.
- **Fluidité Extrême** : Optimisation GSAP & Lenis pour un scroll 60 FPS constant.
- **Sécurité "By Design"** : Protection contre le scraping et détection de bots automatisés.
